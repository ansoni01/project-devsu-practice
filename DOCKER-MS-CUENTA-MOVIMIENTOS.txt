* Para correr la aplicación en docker ejecutamos los siguientes comandos:

docker build -t ms-cuenta-movimientos-docker .
docker run --name ms-cuenta-movimientos-docker -p 8081:8081 ms-cuenta-movimientos-docker:latest

esto permitira levantar el proyecto cuenta movimientos en el puerto 8081

* Como paso siguiente, importar en el postman la collección para realizar las pruebas el archivo
 DEVSU.postman_collection
